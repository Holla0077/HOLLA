# HOLLA project instructions

This is an in-progress fintech web app.

## Rules
- Do not redesign the existing UI theme.
- Preserve the blue-black background and emerald accent color system.
- Reuse current layout and components where possible.
- Do not remove existing API routes unless replacing with a compatible improvement.
- Do not change Prisma model names without updating all dependent code.
- Maintain App Router structure.
- Prefer incremental fixes over rewrites.

## Current priorities
1. Settings route and profile save
2. Verification gating
3. Transaction POST route
4. Wallet and transaction integration
